How to Contribute

We welcome contributions from developers, researchers, and non-technical collaborators.

Developer Workflow

  1. Fork the repository on GitHub.
  2. Clone your fork and create a feature branch.
  3. Commit your changes with clear messages.
  4. Run pytest then open a pull request.

Non-technical Contributions

  1. Browse open issues and documentation.
  2. Suggest new constructs or datasets via GitHub Issues.
  3. Join discussions and provide feedback on methodology.
  4. Please follow the Code of Conduct.
